## Capabilities

`Svelte` `SvelteKit` `React` `TailwindCSS` `CSS` `GSAP` `Design Systems` `UI Motion` `Creative Frontend` `Digital Art`

I enjoy building interfaces that feel intentional from the first impression down to the smallest transition: component systems with personality, landing pages with narrative flow, and interactions that support the design instead of decorating it.

## Approach

- Design with mood, then refine with structure.
- Use motion to guide attention, not compete for it.
- Build components that stay expressive without becoming fragile.
